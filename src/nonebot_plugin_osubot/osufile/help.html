OSUBOT 使用指南

先绑定，再查询。大多数命令都支持指定玩家、模式与 Mods。
nonebot_plugin_osubot
三步开始
1
/bind 玩家

支持用户名、UID 或个人主页，例如 /bind peppy

2
/mode o / t / c / m

设定默认模式；不带参数可查看当前模式

3
/info 或 /bp

开始查询自己的资料与成绩

模式与参数
o / 0 osu!t / 1 taiko c / 2 catchm / 3 mania

临时查别人：/info peppy:0

带 Mods 查询：/bp 5 +HDHR

查询范围：/bl 5-10

复用上一张谱面：/m 782309 后直接发送 /sc

非 std 谱面自动使用原生模式；SB RX 自动匹配分类

SB 服：先 /sbbind 玩家，查询末尾加 &sb

支持 osu! 链接；反向范围 30-1 会自动纠正

osu!

OSUBOT 完整命令

方括号表示可选项，尖括号表示必填项。
COMMAND REFERENCE
通用查询语法
/命令 [玩家] [序号/范围]:[模式] [+Mods] [&sb]
示例：/bl peppy 5-10:o +HDHR　支持 osu! 链接，反向范围会自动纠正。
模式名称与简称
o / 0 osu!t / 1 taikoc / 2 catchm / 3 mania
未填写时使用绑定账号中保存的默认模式。
ppysb 服务器查询
/sbbind <玩家>绑定 ppysb 账号
/info &sb查询已绑定 SB 玩家资料
/bl:4 &sb查询 RX std BP 1–30
/rl:5 &sb查询 RX taiko 最近游玩
/sc <mapid>:6 &sb查询 RX catch 谱面成绩
/sbunbind解除 ppysb 账号绑定
模式 0 / 1 / 2 / 3std / taiko / catch / mania
模式 4 / 5 / 6 / 8RX std / RX taiko / RX catch / AP std
账号与资料
/bind <玩家>绑定用户名、UID 或主页链接；纯数字优先视为用户名
/unbind解除账号绑定
/info [玩家]:[模式]查询玩家资料
/mu发送已绑定玩家主页
/mode [o/t/c/m]查看或修改默认模式；也支持全名及数字 0-3
/lazer切换 stable / lazer 成绩；可简写 /lz
/update刷新个人信息缓存
成绩查询与分析
/bp [序号]查询单条最佳成绩
/bl [起始-结束]BP 列表；默认 1–30
/re [序号]单条最近游玩，包含未通过
/rl [起始-结束]最近游玩列表；默认 1–30
/pr [序号]单条最近通过成绩
/pl [起始-结束]最近通过列表；默认 1–30
/sc <mapid>查询指定谱面成绩（/score）
/nb [#天数]查询最近新增 BP（/tbp）
/bpa [玩家]:[模式]分析 BP 数据分布
/hs [#天数]查看 PP / 排名变化（/history）
谱面工具
/m <mapid> [+Mods]单张难度信息（/map）
/bm <setid>谱面集信息（/bmap）
/预览 <mapid>:[模式]生成普通谱面预览图
/预览 <mapid> +gif生成约 10 秒的动态预览
/完整预览 <mapid>生成完整预览图片
/vp <mapid>生成完整预览视频（/视频预览，耗时较长）
/bg [mapid]提取谱面背景（/getbg）；留空复用最近谱面
/dl [setid]下载谱面并上传群文件（/osudl）
/倍速 <mapid> <速率>生成 mania 倍速谱面
/反键 <setid> [间距] [阈值]生成 mania 反键谱面
/cv --help高级 mania 谱面转换（/convert）
个性化与扩展
/推荐 [玩家]:[模式]获取个性化谱面推荐
/md <成就名>查询成就达成方式（/medal）
/mp <matchid>查看多人比赛结果（/match）
/rt <matchid>计算多人比赛 rating（/rating）
/rank查看本群 PP 排名（/群内排名）
猜歌小游戏
/音频猜歌开始音频猜歌
/音频提示获取当前音频题提示
/图片猜歌开始背景图片猜歌
/图片提示获取当前图片题提示
/谱面猜歌开始谱面片段猜歌
/谱面提示获取当前谱面题提示
输入命令查帮助：/oh bp · /oh vp · /oh bg · /oh rank返回快速指南：/oh
